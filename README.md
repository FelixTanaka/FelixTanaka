# 💫 About Me:
🚀 Halo, Saya Felix Tanaka!<br><br>Bagi saya, menulis kode bukan sekadar memberikan instruksi ke komputer, melainkan cara kita menciptakan solusi untuk masalah di dunia nyata. Ketertarikan saya di bidang *software development* selalu bertumbuh setiap kali melihat bagaimana database yang terstruktur rapi dan *user interface* yang mulus bisa mempermudah kehidupan sehari-hari orang lain.<br><br>Sebagai mahasiswa Informatika, saya fokus mendalami *full-stack development*. Saya tertantang untuk membangun sistem *backend* yang andal dan aman menggunakan PHP dan MySQL. Namun, saya juga sangat peduli dengan kesan pertama pengguna, itulah mengapa saya senang menghidupkan antarmuka web maupun mobile lewat React dan Tailwind CSS. Saya juga terbiasa menerapkan metodologi pengembangan yang terstruktur agar setiap project yang saya kerjakan memiliki kualitas yang baik.<br><br>Saat ini, project utama yang sedang saya bangun adalah sebuah aplikasi mobile marketplace makanan sehat untuk pasar di Indonesia. Saya selalu bersemangat untuk mencari peluang belajar baru, berkolaborasi dalam project web/mobile, dan berkembang bersama para engineer berpengalaman di industri ini.<br><br>
